# Plant Disease Detection using CNN (VGG16)

## Project Overview

This project is a deep learning-based Plant Disease Detection system developed using Convolutional Neural Networks (CNN). The objective of the project is to classify plant leaf diseases using image data in order to assist in early disease identification and improve crop management.

The project was developed as part of a Data Science / Deep Learning assignment focused on image classification and real-world AI applications.

---

## Objectives

* Perform multi-class image classification
* Understand CNN-based image processing
* Apply VGG16 architecture for classification
* Train and evaluate a deep learning model
* Handle a large image dataset
